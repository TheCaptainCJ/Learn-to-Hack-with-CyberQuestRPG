# CyberQuest
CyberQuest Academy is a Python-based cybersecurity RPG that teaches Linux, networking, and hacking concepts through interactive, hands-on missions.

Developed a Python-based cybersecurity training application that gamifies learning through interactive missions covering Linux, networking, and security concepts. Implemented a progression system with skill trees, XP tracking, and persistent user data to simulate real-world cyber operations in a structured learning environment.
